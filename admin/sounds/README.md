# Notification Sounds

This directory should contain notification sound files:

- `notification.mp3` - MP3 format notification sound
- `notification.ogg` - OGG format notification sound (fallback)

## How to add sounds:

1. Download or create a short notification sound (0.5-2 seconds)
2. Convert it to both MP3 and OGG formats
3. Place the files in this directory

## Free sound resources:

- https://notificationsounds.com/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/notification/

## Note:

If sound files are missing, the system will automatically fall back to using the Web Audio API to generate a beep sound.
