### Common Questions

**Q: Can I change an admin-set price?**
A: No, admin-set prices are locked for consistency. Contact admin if you think price needs adjustment.

**Q: What if actual work costs more than admin price?**
A: Contact admin immediately. Explain the situation. Admin may adjust the price or approve additional charges.

**Q: How do I know if I can set the price?**
A: Check the badge:
- üîí Admin Set = Cannot change
- ‚úèÔ∏è Flexible = You can set

**Q: What price should I set for flexible services?**
A: Consider:
- Materials used
- Time spent
- Complexity of work
- Market rates
- Fair value for customer

**Q: Can I see price history?**
A: Currently no, but you can check the Service Prices page anytime for current rates.
